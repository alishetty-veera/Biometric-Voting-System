# Biometric Voting System

## Project Overview

The Biometric Voting System is an embedded systems project designed to improve the security and reliability of the voting process through fingerprint-based voter authentication. The system verifies a voter's identity using a fingerprint sensor before allowing them to cast a vote. This approach helps prevent unauthorized access, duplicate voting, and electoral fraud.

The project demonstrates the integration of biometric authentication with embedded hardware and software to create a secure electronic voting solution.

## Objectives

* Ensure secure voter authentication using biometric data.
* Prevent multiple votes from the same voter.
* Improve voting accuracy and transparency.
* Demonstrate the application of embedded systems in real-world security solutions.

## Features

* Fingerprint-based voter identification and verification.
* Prevention of duplicate voting.
* Secure vote recording.
* User-friendly voting interface.
* Real-time vote counting.
* Visual and audio feedback for user actions.
* Low-cost and efficient embedded implementation.

## Hardware Components

| Component                       | Purpose                  |
| ------------------------------- | ------------------------ |
| Arduino Uno                     | Main controller          |
| Fingerprint Sensor (R305/AS608) | Biometric authentication |
| LCD Display (16x2)              | Display system messages  |
| Push Buttons                    | Candidate selection      |
| Buzzer                          | Audio indication         |
| LEDs                            | Status indication        |
| Breadboard & Jumper Wires       | Circuit connections      |
| Power Supply                    | System power source      |

## Software Requirements

* Arduino IDE
* Embedded C Programming
* Fingerprint Sensor Library
* Serial Monitor for debugging and testing

## System Workflow

1. Voter places a finger on the fingerprint sensor.
2. The system verifies the fingerprint against stored records.
3. If authentication is successful, voting options are displayed.
4. The voter selects a preferred candidate using push buttons.
5. The vote is securely recorded.
6. The system prevents the voter from voting again.
7. Vote counts are updated and stored.
