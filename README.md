# Biometric Voting System

## Overview
A secure electronic voting system that uses fingerprint authentication to verify voters and prevent duplicate voting.

## Features
- Fingerprint-based authentication
- One vote per voter
- Secure vote recording
- Real-time vote counting

## Components Used
- Arduino Uno
- Fingerprint Sensor
- LCD Display
- Push Buttons
- Buzzer
